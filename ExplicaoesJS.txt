Vai criar um documento, que vai ser selecionado do arquivo, vai fazer uma query (consulta) e procurar O
elemento, para utilizar no js, com base em um seletor CSS, como pela classe, ou seção que você buscaria
no css, exemplo: 

document.querySelectorAll("#inicio")

document.querySelectorAll("#clinica")

document.querySelectorAll(".duvida")

document.querySelectorAll(".button")

Diferença no document.querySelectorAll para o document.querySelector 
é que o document.querySelectorAll vai pegar todos os elementos que se encaixarem com aquela Query, já o 
document.querySelector vai pegar apenas o primeiro

//Aqui estou declarando uma variável que vai receber o resultado da Query .duvida, no queryselectorll,
ou seja todos os elementos da classe duvida vão ser atribuidos a essa variável.

let elementosDuvida = document.querySelectorAll('.duvida')

//Aqui estou adicionando uma estrutura forEach (É como se fosse um for, mas o invés de instanciar dentro
do loop as variaveis de contador, ele vai executar o loop sobre um array) para essa variável aonde 
teremos um .addEventListener, ele vai ficar atento a ocorrência de algum evento para desencadear alguma 
ação, no caso passamos o 'click' como evento para desencadear essa ação, que por sua vez será uma função
que criará a classe fundo-verde dentro aonde houver a classe duvidas, então adicionei o fundo-verde no css
para testar e ao clicar o background color vai ficar verde de fato.


// de forma simples o forEach está dizendo que eu quero executar uma função para cada elemento duvida
que escolhi na la em cima

elementosDuvida.forEach(function (duvida){
  duvida.addEventListener('click', function(){
    duvida.classList.add('fundo-verde')
  })
})
Então o trecho de código completo fica: 

let elementosDuvida = document.querySelectorAll('.duvida')
elementosDuvida.forEach(function (duvida){
  duvida.addEventListener('click', function(){
    duvida.classList.add('fundo-verde')
  })
})

Para cada seção de dúvida será adicionado um evento de clique, esse evento de clique vai adicionar
a classe.
Esse exemplo serviu apenas para verificar como essas funções funcionarão na prática, no site usaremos
outra função


A gente vai fazer a mesma coisa agora pra adicionar ou tirar a resposta da dúvida escolhida, mas para isso usaremos o toggle ao invés do
add, porque com o toggle se vc clicar novamente ele volta a especificação anterior, basicamente ele tira a classe se ela ja existir, 
ou adicionando a classe se ela não existir.



let elementosDuvida = document.querySelectorAll('.duvida')

elementosDuvida.forEach(function (duvida){
  duvida.addEventListener('click', function(){
    duvida.classList.toggle('ativa')
  })
})
